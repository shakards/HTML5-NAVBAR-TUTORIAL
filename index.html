Basic Html Navbar Tutorial

This is a tutorial for creating a Basic HTML navbar.

What is a Navbar?

A navbar (short for navigation bar) is a key element in web design that provides users with a simple and consistent way to navigate between different sections or pages of a website.

Typically positioned at the top of a webpage, the navbar contains links or buttons labeled with page names such as Home, About, Services, and Contact.

In HTML, a navbar is commonly created using a combination of the <nav> or <div> tags, along with anchor (<a>) tags for links. CSS is then used to style the navbar and make it visually appealing, ensuring it aligns with the overall theme of the website.

Why Use a Navbar?

  • Improved User Experience: A well-structured navbar helps visitors find content quickly and easily.
  • Website Organization: It organizes the site's layout, making it intuitive to navigate.
  • Consistency: A navbar is present on every page, ensuring uniformity across the site.

Below is the code used to create the above navbar. Click the "Copy" button to copy it to your clipboard.


        <!DOCTYPE html>
        <html lang="en">
        <headmeta charset="UTF-8">
          <meta name="viewport" content="width=device-width, initial-scale=1.0">
          <title>HTML Navbar</title>
          <style>
            .navbar {
              display: flex;
              background-color: #333;
              padding: 10px 20px;
            }
            .navbar .logo {
              font-size: 20px;
              font-weight: bold;
              color: white;
              margin-right: auto;
            }
            .navbar a {
              color: white;
              text-decoration: none;
              padding: 10px 15px;
              margin: 0 5px;
            }
            .navbar a:hover {
              background-color: #575757;
              border-radius: 5px;
            }
          </style>
        </head>
        <body>
          <div class="navbar">
            <div class="logo">MyWebsite</div>
            <a href="#home">Home</a>
            <a href="#about">About</a>
            <a href="#services">Services</a>
            <a href="#contact">Contact</a>
          </div>
        </body>
        </html>

  • <!DOCTYPE html>: This declaration is essential as it tells the browser that this document is written in HTML5. Without this, the browser might misinterpret the page, especially when using modern HTML features.

  • <html lang="en">: Specifies the root element of the HTML document. The
    lang="en" attribute indicates that the page's language is English. Helpful for search engines and assistive technologies like screen readers.


  • <head>: Contains metadata such as the page title and links to stylesheets or scripts. Does not display content directly on the page but is crucial for functionality and SEO.


  • <body>: The main container for the visible content of the webpage. Everything displayed to users (e.g., text, links, images) is written here.


  • <div class="navbar">: A element used as a container for the navigation bar.
    This class helps apply CSS styles to structure and design the navbar.


  • <a>: Represents a hyperlink for navigating to other sections or pages. Key attributes include href, which specifies the destination URL.

    HTML Navbar with Icons Tutorial

    This is a tutorial for creating a navbar in HTML with icons next to the links.

    What is a Navbar?

    A navbar (short for navigation bar) is a key element in web design that provides users with a simple and consistent way to navigate between different sections or pages of a website.

    Typically positioned at the top of a webpage, the navbar contains links or buttons labeled with page names such as Home, About, Services, and Contact, often enhanced with icons.

    In HTML, a navbar is commonly created using the <nav> or <div> tags, with anchor (<a>) tags for links. We can also use icons from libraries like Font Awesome for a more visual appeal. CSS is used to style the navbar, making it more user-friendly and visually aligned with the overall theme of the website.

    Why Use Icons in a Navbar?

    • Enhanced Visual Appeal: Icons make the navbar look more engaging and modern.
    • Improved Usability: Icons can help users quickly identify the functionality of each link.
    • Consistent User Interface: Icons add a cohesive design element that can be used throughout the website.

    How to Create a Navbar with Icons?

    Here’s how you can create a simple HTML navbar with icons using Font Awesome.

            <!DOCTYPE html>
        <html lang="en">
        <headmeta charset="UTF-8">
          <meta name="viewport" content="width=device-width, initial-scale=1.0">
          <title>Navbar with Icon</title>
          <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0-beta3/css/all.min.css">
          <style>
            .navbar1 {
              display: flex;
              align-items: center;
              background-color: #333;
              padding: 10px 20px;
            }
            .navbar1 .logo1 {
              font-size: 20px;
              font-weight: bold;
              color: white;
              margin-right: auto;
            }
            .navbar1 a {
              color: white;
              text-decoration: none;
              padding: 10px 15px;
              margin: 0 5px;
              display: flex;
              align-items: center;
            }
            .navbar1 a:hover {
              background-color: #575757;
              border-radius: 5px;
            }
            .navbar1 i {
              margin-right: 8px;
            }
          </style>
        </head>
        <body>
          <div class="navbar1">
            <div class="logo1">MyWebsite</div>
            <a href="#home"><i class="fas fa-home"></i>Home</a>
            <a href="#about"><i class="fas fa-info-circle"></i>About</a>
            <a href="#services"><i class="fas fa-cogs"></i>Services</a>
            <a href="#contact"><i class="fas fa-envelope"></i>Contact</a>
          </div>
        </body>
        </html

  • <!DOCTYPE html>: This declaration is essential as it tells the browser that this document is written in HTML5. Without this, the browser might misinterpret the page, especially when using modern HTML features.

  • <html lang="en">: Specifies the root element of the HTML document. The
    lang="en" attribute indicates that the page's language is English. Helpful for search engines and assistive technologies like screen readers.


  • <head>: Contains metadata such as the page title and links to stylesheets or scripts. Does not display content directly on the page but is crucial for functionality and SEO.


  • <link rel="stylesheet" "stylesheet" href=""https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0-beta3/css/all.min.css": This link loads the Font Awesome library to display icons in the navbar. Icons are embedded using the Font Awesome class names.


  • <style>: Contains the CSS rules to style the navbar and other elements on the page. Here, we define the background color, text styling, and hover effects for the navbar.


  • <body>: Contains the visible content of the page, including the navbar and other sections. The body holds all elements that users will interact with on the webpage.


  • <div class="navbar1">: A container for the navbar links, using flexbox to align items horizontally. Flexbox allows for easy alignment and distribution of navbar items.


  • <a>: Represents a hyperlink for navigating to other sections or pages. Each a tag contains an icon (using the i tag) and text. The href attribute links to the corresponding section.
